# Breakout Android App (Versión Alfa)

## Descripción

Aplicación móvil desarrollada en **Android Studio**, basada en el clásico juego *Breakout*. Este proyecto fue creado para practicar desarrollo móvil, lógica de programación y diseño de interacción con el usuario.

##  Funcionalidades

* Movimiento de la barra controlado por el usuario
* Sistema básico de colisiones
* Lógica de rebote de la pelota
* Estructura base de videojuego en Android

##  Tecnologías utilizadas

* Java
* Android Studio
* Desarrollo móvil nativo

## Estado del proyecto

 Versión Alfa
El proyecto se encuentra en fase inicial, enfocado en la implementación de mecánicas básicas.

Objetivo del proyecto

Aplicar conocimientos de:

* Desarrollo de aplicaciones móviles
* Programación orientada a objetos
* Lógica de videojuegos
